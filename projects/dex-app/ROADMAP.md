# ROADMAP.md - DEX App

## Vision
Mobile-first Next.js application designed to crush the 'Collector' app via superior vision logic.

## Core Protocol: Batch Array
- Phase 1: Grid-detection implementation. Multi-boundary identification in single high-res images.
- Phase 2: Parallel OCR extraction. Mapping 10+ cards/IDs into a single JSON array pass.
- Phase 3: Automated data verification against global databases.
